# Plant Monitoring App

This is a **Flutter-based Plant Monitoring application** that tracks the moisture levels of various plants and provides **real-time alerts** when moisture levels fall below a certain threshold. The app ensures plants are well-watered and taken care of through proactive monitoring and voice alerts. üöÄ
